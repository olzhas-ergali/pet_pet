# OptBirja - Живая оптовая биржа товаров

## 🎯 Описание продукта

Современная платформа для оптовых закупок с real-time обновлением цен, где поставщики могут менять цены в режиме реального времени, а покупатели видят изменения мгновенно и могут успеть купить по выгодной цене.

## ✨ Основные возможности

### 🔥 Real-time обновление цен
- Цены обновляются каждые 6-8 секунд
- Анимированное отображение изменений
- Мигающий эффект при падении/росте цены
- График истории изменения цен

### 💎 FOMO-механики
- **"Кто-то только что купил"** - всплывающие уведомления о покупках
- **"Сегодня дешевле"** - товары с максимальными скидками
- **"Цена упала"** - товары, где цена недавно снизилась
- **"Быстро раскупают"** - товары с низким остатком

### 📊 Система оптовых цен
- 3 уровня оптовых цен (1-10, 10-50, 50+)
- Автоматический расчет скидки при увеличении количества
- Визуальная индикация выгоды

### 🎨 Премиум-дизайн
- Современный UI в стиле Kaspi/Arbuz/Glovo
- Типографика Inter
- Мягкие тени и скругления (12-16px)
- Градиенты и анимации
- Skeleton loading
- Адаптивный дизайн

## 📱 Экраны

### 1. Авторизация (`/auth`)
- Вход по номеру телефона
- OTP-код подтверждения
- Минимум полей, максимум удобства

### 2. Главная страница (`/`)
- Hero-баннер с градиентом
- Секции "Сегодня дешевле", "Цена упала", "Быстро раскупают"
- Фильтр по категориям
- Сетка товаров с анимациями

### 3. Каталог (`/catalog`)
- Поиск товаров
- Фильтры (категория, цена, поставщик)
- Адаптивная сетка товаров

### 4. Карточка товара (`/product/:id`)
- Галерея изображений
- Real-time цена с анимацией
- График истории цен
- Система оптовых уровней
- Выбор количества
- Информация о поставщике и доставке

### 5. Корзина (`/cart`)
- Список товаров
- Автоматический расчет оптовой скидки
- Подсказки о дополнительной выгоде
- Бесплатная доставка от 10,000 ₸

### 6. Профиль (`/profile`)
- Статистика покупок
- Настройки аккаунта
- Адреса и способы оплаты

## 🛠 Технологии

- **React 18** - UI framework
- **TypeScript** - типизация
- **React Router** - навигация
- **Tailwind CSS v4** - стилизация
- **Motion (Framer Motion)** - анимации
- **Recharts** - графики
- **date-fns** - работа с датами
- **Lucide React** - иконки
- **Sonner** - toast-уведомления

## 🔥 Ключевые UX-фишки

1. **Анимации**
   - Hover-эффекты на карточках
   - Плавные переходы между экранами
   - Анимация изменения цен
   - Skeleton loading при загрузке

2. **Real-time обновления**
   - Симуляция WebSocket обновлений
   - Мгновенное отображение изменений
   - Toast-уведомления о падении цен

3. **FOMO-механики**
   - Счетчики остатков
   - Таймеры последнего обновления
   - Уведомления о покупках других пользователей
   - Визуальные индикаторы скидок

4. **Адаптивность**
   - Desktop: сетка 4 колонки
   - Tablet: сетка 2-3 колонки
   - Mobile: 1 колонка + нижняя навигация

## 🚀 Следующие шаги для масштабирования

### Backend (Supabase)
1. PostgreSQL для хранения:
   - Товары
   - Заказы
   - Пользователи
   - История цен

2. Realtime для:
   - WebSocket обновления цен
   - Pub/Sub оповещения

3. Storage для:
   - Изображения товаров
   - Документы

### Интеграции

2. **WhatsApp** уведомления клиентам
3. **Kaspi** интеграция оплаты
